MinePinger#7640 is a bot to view the status of any Minecraft server.
Invite the bot from this link: https://bit.ly/minepinger

Main command:
- ?status <IP address>
- ?help
- ?info

Here's what you can see about a server:
- Address
- Port
- Status
- Latency
- SRV Record
- Minecraft Version
- Players
- MOTD
